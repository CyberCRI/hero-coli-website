Detailed Description

This class represent all the reactions that are instantaneous The reactions that look like :

2H + O = H2O should be managed by this class. see react() method for more details.

override void InstantReaction.react ( ArrayList  molecules)
inlinevirtual

This function is called at each frame. It find the limiting reactant and consume as reactant and produce product as much as possible.

The formula is : 

 delta =  Min(Reactant_1 / Coef_1, Reactant_2 / Coef_2, ... , Reactant_n / Coef_n)
 for each product P : [P] += delta * Coef_P
 for each reactant R : [R] -= delta * Coef_R
