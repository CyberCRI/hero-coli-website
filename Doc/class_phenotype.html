Detailed Description

This class represent the interface between the Reaction Engine and the game.

Each class that use molecules concentration to perform a graphical effect should inherit from this class. This abstract class implement the basics stuff that a phenotype needs. It ask to the reaction engine the concentration of each protein that of the medium that it represent.

Some attribute have to be define before runing it : 

          - A reference to the reaction engine
          - A mediumId

This class also implement so usefull functions like a hill function.
