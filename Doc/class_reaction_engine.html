Detailed Description

The main class that compute all the reactions.

This class initialize from files and execute all the reactions. The reactions that are currently implemented are :

  • Degradation
  • Diffusion (Fick)
  • Enzyme reaction with effectors (EnzymeReaction)
  • Promoter expressions (Promoter)
