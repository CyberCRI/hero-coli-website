Detailed Description

This class load all the allostery reactions.

A allostery's reaction's declaration should respect this synthax : 

<allostery>
  <name>inhibitLacI</name>
  <effector>IPTG</effector>
  <EnergyCost>0.3</EnergyCost>
  <K>0.1</K>
  <n>2</n>
  <protein>LacI</protein>
  <products>LacI*</products>
</allostery>
